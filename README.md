# dotfiles

## How to get started

1. Install the apps

    ```bash
    sh install/install.sh
    ```

2. Set up the apps

    ```bash
    sh install/setup.sh
    ```

3. Set up login items in Preferences/Users & Groups
4. Move MacOS preference files

    ```bash
    cp ./system/* /Library/Prefrences/
    ```
