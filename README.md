# dotfiles

## How to get started

### MacOS

1. Run shell files

    ```bash
    source ./install/mac.sh
    ```

2. Move MacOS preference files

    ```bash
    cp ./system/* /Library/Prefrences/
    ```

### WSL2

```bash
source install/wsl.sh
```
